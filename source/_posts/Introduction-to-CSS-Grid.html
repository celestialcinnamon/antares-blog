Ang CSS Grid ang pinakamalakas na layout tool na mayroon tayo sa Web ngayon. Sa artikulong ito, tingnan natin ang mga konsepto sa likod ng CSS Grid at kung paano mo ito magagamit bilang pamalit sa UI framework na mayroon ka.

Basic Concepts

Dahil sa CSS Grid, nagkaroon ang CSS ng two-dimensional at tunay na grid. Magagamit ang mga ito para mag-layout ng malalaking bahagi ng isang page o ng maliliit na user interface elements.

Ano ang grid?

Ang isang grid ay set ng mga horizontal at vertical lines na nagtatagpo sa isang point. Dahil sa mga intersections na ito, nagkakaroon ng rows at columns.

Track sizes

Ang mga track ay tumutukoy sa column o sa row. Puwede kang mag-set ng fixed-width na track. Puwede ring flexible, nagre-resize depende sa size ng viewport o sa size ng laman ng track.

Item placement

Puwede mong ilagay sa eksaktong lokasyon ang mga items sa grid gamit ang mga line number o area names. Puwede ring hayaan mong ang browser ang mag-layout para sa iyo.

Additional tracks

Puwede mong i-set kung ilang columns o rows ang mayroon sa grid. Puwede ring kusang magdagdag ang browser para sa iyo.

Alignment

Kontrolado mo ang lahat ng alignments sa grid.

Item placement

Puwede kang maglagay ng higit sa isang item sa isang grid cell o area, at puwede mo silang pagpatungin gamit ang z-index property.

Ang Grid Container

We create a grid container by declaring display: grid or display: inline-grid on an element. As soon as we do this all direct children of that element will become grid items.

    <div class="wrapper">
      <div>One</div>
      <div>Two</div>
      <div>Three</div>
      <div>Four</div>
      <div>Five</div>
    </div>
  
    .wrapper {
      display: grid;
    }

All the direct children are now grid items. In a web browser, you won’t see any difference to how these items are displayed before turning them into a grid, as grid has created a single column grid for the items. At this point, you may find it useful to work with the Grid Inspector, available as part of Firefox's Developer Tools. If you view this example in Firefox and inspect the grid, you will see a small icon next to the value grid. Click this and then the grid on this element will be overlaid in the browser window.
